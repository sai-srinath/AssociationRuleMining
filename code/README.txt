The program can be run on any Java IDE.
The input file is given as a command line arguement.
The program initially asks for the minimum support and confidence values to be input by the user. The user is required to input these values one line after another (in %).
eg.Enter the support and confidence(in %)
50
70
The program will output the size of every frequent itemset followed by the sum of all frequent itemsets.
It will then ouput all the generated rules followed by the number of such generated rules.
